# Theme system

## Modes

- `Light` is the first-launch default and the primary Luminous Workshop presentation.
- `Dark` is a complete semantic theme rather than a tinted or inverted skin.
- `System` is available only as an explicit user choice and is remembered with other non-sensitive preferences.
- The former Graphite, Mirage, and Ember decorative backgrounds do not carry into 3.0.

## Semantic color

| Role | Visual family | Use |
| --- | --- | --- |
| Primary | Sea-glass cyan | Main action and active processing |
| Relation | Soft lilac | Selection, association, and contextual focus |
| Attention | Warm amber | Needs Attention and non-destructive warnings |
| Danger | Restrained red | Destructive or irreversible actions only |
| Foundation | Warm neutral / graphite | Surfaces, text, dividers, and quiet depth |

Color never carries status by itself. Text, iconography, shape, or position provides a second signal.

## Accessibility and motion

- Normal text and interactive controls meet WCAG AA contrast targets in both themes.
- Keyboard focus remains visible and is not replaced by hover treatment.
- Reduced-motion preference removes non-essential transitions and all ambient movement.
- Motion is reserved for source arrival, task-stage change, and deliverable completion.
- Background glow, perpetual shimmer, and decorative floating elements are excluded.
