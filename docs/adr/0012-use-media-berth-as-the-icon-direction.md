# Use Media Berth as the icon direction

Media Dock 3.0 uses the selected “Media Berth” concept—two protective interlocking forms around one warm deliverable tile—as the icon direction because it expresses docking, transformation, and safekeeping without relying on a play symbol or tool logo. The selected generated image at `docs/design/assets/media-dock-3-media-berth-direction.png` is a direction master rather than a shipping asset; production variants must simplify highlights for small sizes, resolve the opaque corner background, and preserve the berth silhouette across macOS, Windows, and favicon masks.
