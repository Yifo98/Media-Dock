# Compose tasks from the source first

Media Dock 3.0 creates work through one Source Dock that accepts links, files, and folders, identifies the source before creating a task, and then offers only compatible deliverables with a recommended default. Users no longer choose a downloader or media tool before the product understands their material; unsupported sources, access requirements, and incompatible combinations are explained before execution, while technical format and runtime controls remain in the expert inspector.
