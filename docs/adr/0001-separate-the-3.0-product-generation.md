# Separate the 3.0 product generation

Media Dock 3.0 is a new product generation rather than a visual patch on the 2.1 series. The published `v2.1.2` tag, release assets, and stable behavior remain intact; future 2.1 work is limited to necessary maintenance, while the interface, product architecture, and brand redesign are developed on an isolated 3.0 line so that the new model can evolve without destabilizing the current release.
