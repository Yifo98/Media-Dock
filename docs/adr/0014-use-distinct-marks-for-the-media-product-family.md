# Use distinct marks for the media product family

Media Dock and MediaCookies share the Luminous Workshop palette, material restraint, geometry discipline, and small-size clarity while retaining distinct product marks: Media Dock uses the compact Media Berth, and MediaCookies uses its Passage Mark. Brand assets are coordinated through documented visual principles rather than shared source files, allowing both products to evolve and ship in parallel without confusing the desktop application with the browser extension.
