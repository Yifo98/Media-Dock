# Keep the Media Dock product name

The 3.0 generation keeps the Media Dock name while rebuilding its icon and brand expression around the new local-media-workspace model. “Dock” now describes sources arriving, media tasks being processed, and deliverables leaving, preserving repository, updater, MediaCookies, and user-recognition continuity without tying the product name to yt-dlp, downloading, or one tool category.
