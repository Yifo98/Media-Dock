# Center the product on media tasks

Media Dock 3.0 is organized around the lifecycle of a media task: a user supplies a network or local source, Media Dock identifies and acquires it, applies the requested processing, and delivers a usable result. Downloading, merging, extraction, subtitles, credentials, runtimes, and logs remain capabilities of that lifecycle rather than competing top-level products, giving the interface one stable mental model as new capabilities are added.
