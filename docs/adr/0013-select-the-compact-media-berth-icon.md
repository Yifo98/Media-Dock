# Select the compact Media Berth icon

The compact third refinement is the selected Media Dock 3.0 icon master because its flatter geometry, stronger contour contrast, and reduced depth preserve the Media Berth identity at taskbar and favicon sizes. The selected visual master is stored at `docs/design/assets/media-dock-3-icon-selected.png`; it is not yet a shipping export because the generated checkerboard is baked into an opaque image, so final platform assets must reconstruct clean transparency, safe margins, and size-specific rendering without changing the chosen mark.
