# Use four product spaces and a contextual inspector

Media Dock 3.0 uses one desktop window with four stable product spaces: Workbench, Task Center, Deliverable Library, and System Center. A contextual expert inspector opens on demand for the selected task, while download, merge, extraction, and subtitle capabilities enter through media-task creation rather than competing as top-level workspaces; this prevents runtime status, logs, history, configuration, and creative work from sharing one permanent dashboard.
