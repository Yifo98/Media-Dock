# Use actionable task states and explicit stages

Media tasks expose five user-facing states—Queued, Running, Needs Attention, Completed, and Cancelled—while active work reports Preparing, Acquiring, Processing, or Delivering as its current stage. Technical failures and interruptions become actionable Needs Attention reasons instead of terminal generic errors, and acquisition reaching 100 percent never implies completion while processing or delivery remains unfinished; raw tool details stay in the expert inspector.
