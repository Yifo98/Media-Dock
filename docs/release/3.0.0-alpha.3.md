# Media Dock 3.0.0 Alpha 3 — YouTube Mix duplicate-task repair

This unsigned developer preview is an internal Windows test candidate. It is not a general-availability release.

## What changed

- Repeated videos returned by a YouTube Mix are exposed as one selectable item during source inspection.
- Repeated identical task plans are collapsed at the task-engine boundary before database records are created.
- Different videos, output choices, quality selections, and delivery settings remain independent tasks.

## Test focus

1. Open a YouTube Mix or playlist that previously produced duplicate downloads.
2. Select exactly two different videos.
3. Start processing and confirm that the task center creates exactly two tasks and two final media files.
4. Confirm that the selected Cookie state and available quality options still behave normally.

## Known limitation

This candidate is not code-signed. Windows Smart App Control may block it before launch with Event 3077 / `VerifiedAndReputableDesktop`. A successful functional test does not remove that trust limitation.

Media Dock 2.1.2 remains the stable release and is not modified by this candidate.
